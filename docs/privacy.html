💊 MedAlert — Política de Privacidade

Última atualização: junho de 2025

1. Quem somos

O MedAlert é um aplicativo desenvolvido para auxiliar usuários a compartilhar informações médicas de emergência com socorristas e profissionais de saúde, além de oferecer lembretes para a tomada de medicamentos.

Desenvolvedor: Fabio Estevam Andrade (faema44@gmail.com)

2. Quais dados coletamos

O MedAlert pode armazenar, a critério do próprio usuário, as seguintes informações:

  • Nome completo
  • Tipo sanguíneo
  • Data de nascimento
  • Alergias conhecidas
  • Lista de medicamentos em uso (nome genérico, comercial, dose e frequência)
  • Observações médicas
  • Contatos de emergência (nome e telefone)
Nenhum desses dados é enviado a servidores externos. Todas as informações ficam armazenadas exclusivamente no dispositivo do usuário.

3. Como usamos os dados

Os dados são usados apenas para as funcionalidades do próprio app:

  • Exibir uma notificação persistente na tela de bloqueio com a ficha médica do usuário
  • Verificar interações medicamentosas com base em banco de dados local embutido no app
  • Enviar lembretes locais de medicamentos nos horários configurados pelo usuário

4. Compartilhamento de dados

O MedAlert não compartilha, vende, aluga nem transmite dados pessoais a terceiros. Não há integração com serviços de análise, publicidade ou rastreamento.

A notificação da tela de bloqueio é visível por qualquer pessoa com acesso físico ao dispositivo — esse comportamento é intencional, pois o objetivo é permitir que socorristas vejam a ficha médica sem precisar desbloquear o celular.

5. Armazenamento e segurança

Os dados são armazenados em banco de dados SQLite local, protegido pelo sistema operacional Android. O aplicativo não requer conexão com a internet para funcionar.

A segurança dos dados está sujeita às proteções de segurança do próprio dispositivo (senha de desbloqueio, criptografia de disco, etc.).

6. Permissões solicitadas

  • Notificações — para exibir o alerta de emergência e os lembretes de medicamentos
  • Alarmes exatos — para disparar lembretes nos horários precisos configurados pelo usuário

Nenhuma permissão de localização, câmera, microfone ou acesso a contatos do sistema é solicitada.

7. Crianças

O MedAlert não é direcionado a crianças menores de 13 anos e não coleta conscientemente dados de menores. O uso por menores deve ser supervisionado por um responsável.

8. Seus direitos

Como todos os dados ficam no seu dispositivo, você tem controle total sobre eles:

  • Acessar — todas as informações são visíveis dentro do próprio app
  • Corrigir — edite diretamente no app a qualquer momento
  • Excluir — desinstalar o app remove todos os dados permanentemente

9. Alterações nesta política

Eventuais atualizações serão publicadas nesta página com a data de revisão atualizada. Recomendamos verificar esta página periodicamente.
